# 🤖 MCP & 626Labs Integration Guide

This repository is optimized for the **626Labs Dashboard** and the **Model Context Protocol (MCP)**.

## Getting Started

1. **Connect to Dashboard**: In the 626Labs UI, link this repository to your project.
2. **Configure MCP Server**: Ensure your IDE is using the `@626labs/mcp-server` bridge.
3. **Context Injection**: Agents within this repo will automatically receive project briefs and active tasks.

## Key Capabilities

- **`update_project`**: Use this to keep the dashboard informed of your progress.
- **`log_decision`**: Log rationales to ensure session continuity.
- **`get_ecosystem_context`**: Query related apps via tags to understand shared dependencies.

## Standard Workflows

- `npm run deploy`: Automated deployment with health checks.
- `npm run test`: Sandbox verification.
