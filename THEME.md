# 🎨 Theming by Code (TBC) Protocol

This project follows the **Theming by Code** principle. Visual identity is a function of logic, not ad-hoc styling.

## Design Tokens

- **Root-Level CSS Variables**: All colors, spacing, and typography are defined in `:root`.
- **Glossmorphic Surface**: Default UI components should use semi-transparent backgrounds with backdrop filters.
- **Dynamic State**: Hover, Active, and Focus states are implemented as micro-animations.

## Implementation Guide

1. Use `var(--color-primary)` instead of hardcoded hex codes.
2. Use `rem` for accessibility-first typography.
3. Every component must support **Dark Mode** by default via system preference.
